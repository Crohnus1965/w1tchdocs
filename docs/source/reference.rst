
DIS2RBED LUA reference
========================

There are multiple namespaces & types in the current DIS2RBED LUA framework, each with their own set of functions.

Types in LUA Engine
----------------------
:ref:`types`
Types in LUA Engine are defined in the following way:

* Hash (uint64)
* Entity (int)
* Ped (int)
* Player (int)
* Cam (int)
* Blip (int)
* Any (int)
* Vehicle (int)
