Clouds Types
===================================
Clouds types are used to set the clouds in the game.
Current available clouds types are:

* ``Cloudy 01``
* ``RAIN``
* ``horizonband1``
* ``horizonband2``
* ``horizonband3``
* ``Puffs``
* ``Wispy``
* ``Horizon``
* ``Stormy 01``
* ``Clear 01``
* ``Snowy 01``
* ``Contrails``
* ``altostratus``
* ``Nimbus``
* ``Cirrus``
* ``cirrocumulus``
* ``stratoscumulus``
* ``Stripey``
* ``horsey``
* ``shower``
