Headlights colors
===================================
Headlights color IDs are used to set the headlights color of the vehicles in the game.
Current available headlight color IDs are:

* Stock = 0
* White = 1
* Blue = 2
* Light Blue = 3
* Mint Green = 4
* Lime Green = 5
* Yellow = 6
* Light Orange = 7
* Orange = 8
* Red = 9
* Light Pink = 10
* Pink = 11
* Light Purple = 12
* Purple = 13
