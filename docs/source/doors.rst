Doors Index
===================================
Doors IDs are used to close and open doors of the vehicles in the game.
Current available door IDs are:

* Front Left Door = ``-1``
* Front Right Door = ``1``  
* Back Left Door = ``0``
* Back Right Door = ``3``
* Hood = ``4``
* Trunk = ``5``
* Back = ``6``
* Back2 = ``7``
