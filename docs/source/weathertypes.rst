Weather Types
===================================
Weather types are used to set the weather in the game.
Current available weather types are:

* ``CLEAR``
* ``EXTRASUNNY``
* ``CLOUDS``
* ``OVERCAST``
* ``RAIN``
* ``CLEARING``
* ``THUNDER``
* ``SMOG``
* ``FOGGY``
* ``XMAS``
* ``SNOWLIGHT``
* ``BLIZZARD``
* ``NEUTRAL``
* ``SNOW``
* ``HALLOWEEN``
