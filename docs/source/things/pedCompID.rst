Ped Component IDs
====================

* ``0`` -- Head
* ``1`` -- Beard
* ``2`` -- Hair
* ``3`` -- Torso
* ``4`` -- Legs
* ``5`` -- Hands
* ``6`` -- Foot
* ``7`` -- Eyes
* ``8`` -- Accessories like parachute, scuba tank
* ``9`` -- Accessories like bags, mask, scuba mask
* ``10`` -- Decals and mask
* ``11`` -- Auxiliary parts for torso
